## Frontend Development Experience

I develop responsive and interactive user interfaces using modern frontend technologies.

On the web, I work with React, JavaScript, HTML, and CSS to create fast and intuitive applications. For mobile development, I use React Native to build cross-platform applications that deliver a seamless user experience.

One of the projects I contributed to was **Gás Fácil**, a mobile platform that enables users to request and purchase gas through a convenient and user-friendly application. My work involved interface development, API integration, navigation flows, and user experience improvements.
