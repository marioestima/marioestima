## Sobre Mim

Sou um Desenvolvedor Full-Stack focado na criação de aplicações web e mobile modernas, responsivas e escaláveis, com forte inclinação para o desenvolvimento Frontend.

Trabalho construindo interfaces de usuário fluidas, transformando ideias em produtos digitais reais, sem deixar de lado a integração com sistemas backend robustos, APIs e bancos de dados.

Aprecio criar experiências de usuário excepcionais, otimização de performance no lado do cliente e arquitetura limpa.
